Особенности сортировки списка:

Query параметр sort - это строка (если передано одно значение) или массив строк (если передано несколько значений) вида fieldName<пробел>sortDirection

    например: 

    ?sort=avgScores desc&sort=sumScore desc&sort=winsCount desc&sort=lossesCount asc. 

    После преобразования на стороне back-end поле sort из query будет иметь вид: 

    sort: ["avgScores desc", "sumScore desc", "winsCount desc", "lossesCount asc"],

    соответственно, сначала сортируем по полю avgScores направление desc, 

    затем, тех, у кого одиноковые значения по avgScores - сортируем по sumScore desc и т.д.

Если параметр не передан - применяем значение по-умолчанию (см. swagger документацию)

Параметр sort может содержать как одно значение (сортируем только по 1 критерию), так и все возможные значения (сортируем по всем критериям)
